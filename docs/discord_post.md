# Untitled Roguelike RPG


## General Introduction
Turn-Based Combat in the Style of Pokemon / Shin Megami Tensei
Roguelike Progression System in the Style of Slay The Spire / Balatro

Learn & Customize a suite of 'Moves' for Navigating Various Combat Situations
Acquire 'Blessings', Non-Move Effects which Change Game Logic (Think Relics in STS)
Upgrade & Customize Basic Moves with Fragments, Creating completely unique effects
On Death, your build is saved to the game permanently as a Random Encounter for you and others Future Runs


## Game Loop
Setup:
  1. Choose an Initial Build from 4 classes
  2. Customize Starting Build

Main Loop:
  1. Enter Combat
  2. If Win, Gain Resources, else: Permadeath
  3. Upgrade / Customize Build Screen
  4. Provide Path/Special Event Options (If Any Exist)
  5. Repeat
